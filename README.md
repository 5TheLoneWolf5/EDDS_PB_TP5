# Projeto TP4 - Junção dos Projetos Banco CRUD e FinTech

- (TP2) Projeto 1 - CRUD Bancário
- (TP3) Projeto 2 - CRUD FinTech

<b>Como ambos projetos estão conectados:</b> utilizam o mesmo back-end (incluindo o banco de dados), e mostram os mesmos dados no front-end de forma rápida e responsiva.

As informações específicas de cada projeto (como instruções, workflow, coverage, screenshots, etc.) se encontram presentes em seus próprios repositórios. <small>Nota: o build dos projetos em Gradle contém a etapa de testes.</small> 

Foram utilizados os runners hospedados pelo GitHub, por conta facilidade e simplicidade de setup. Isso é diferente da maneira self-hosted, que te dá a liberdade de configurar um ambiente próprio e personalizado (hardware e software) para a execução dos workflows.

O workflow é rodado automaticamente ao fazer push na branch <i>main</i> do back-end.

Foram removidas redundâncias e códigos comentados irrelevantes. Isso deixou o código mais limpo e legível.
